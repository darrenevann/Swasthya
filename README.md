# Swasthya 2026 - Sponsorship Proposal Landing Page

Proyek ini adalah *landing page* web statis yang berfungsi sebagai proposal digital untuk kegiatan Bakti Sosial Desa Binaan BEM UNTAR 2026 ("Swasthya"). Website ini ditujukan untuk calon sponsor dan donatur, memuat informasi latar belakang, *rundown*, paket sponsorship, dan metode pembayaran.

## Informasi Proyek

* **Nama Kegiatan:** Swasthya: Journey Through Adaptation
* **Penyelenggara:** BEM Universitas Tarumanagara
* **Lokasi Target:** Kampung Mangunjaya, Desa Mangunjaya, Sukabumi
* **Waktu Pelaksanaan:** Februari 2026
* **Target Peserta:** 300+ Warga Desa, 200 Penerima Pengobatan, 150 Penerima Sembako

## Teknologi yang Digunakan

* **HTML5** - Struktur konten semantik.
* **CSS3** - *Custom styling* (variabel warna, tipografi, responsivitas).
* **Bootstrap 5.3** - *Framework* CSS untuk *grid system* dan komponen UI (Navbar, Cards, Modal).
* **Google Fonts** - Menggunakan font 'Fredoka One' dan 'Nunito'.
* **Bootstrap Icons** - Ikon antarmuka.
